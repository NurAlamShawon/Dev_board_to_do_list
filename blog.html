Question-1: What are the different ways to select an element in the DOM?

  • document.getElementById("id")
  • document.getElementsByClassName("class")
  • document.getElementsByTagName("tag")
  • document.querySelector("selector")
  • document.querySelectorAll("selector")

Question-2: What is the difference between innerHTML, innerText, and textContent ?

  • innerHTML = Gets or sets HTML content (includes tags).
  • innerText = Gets or sets visible text (ignores hidden elements).
  • textContent = Gets or sets all text, including hidden ones, without formatting.

Question-3: What is event delegation in the DOM?

Instead of adding event listeners to multiple elements, event delegation allows you to add a single listener to a parent element and handle child events efficiently.

Question-4: What is event bubbling in the DOM?

Event bubbling means when an event occurs on a child element, it "bubbles up" and triggers handlers on its ancestors.

Question-5: How do you create, add, and remove elements using JavaScript?

  • to create element : let newDiv = document.createElement("div"); newDiv.textContent = "Hello!";.
  • to add element : document.body.appendChild(newDiv);
  • to remove element : newDiv.remove();
